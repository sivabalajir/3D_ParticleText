A high-performance 3D particle system that morphs **10,000 particles** between a rotating sphere and real-time text.  
Built using **vanilla JavaScript + HTML5 Canvas** — no dependencies, single file.

---

## 🚀 Features
- 🔹 10,000 particles for sharp text rendering  
- 🔹 Real-time typing → particles form text instantly  
- 🔹 Smooth morphing between sphere ↔ text  
- 🔹 True 3D perspective with rotation  
- 🔹 Cursor repulsion interaction  
- 🔹 Multi-line text + responsive scaling  
- 🔹 Retina (HiDPI) support  
- 🔹 Zero dependencies (single HTML file)  

---

## 🎮 Controls
- **Type** → Particles form text  
- **Clear input** → Back to sphere  
- **Mouse move** → Repel particles  
- **Double-click** → Reset to sphere  

---

## ⚡ Tech Highlights
- Typed arrays (`Float32Array`) for high performance  
- 60 FPS real-time rendering  
- Off-screen canvas for text sampling  
- Physics-based particle motion  
- Perspective projection (3D → 2D)
